## 二、类型定义分析

### 2.1 CoordConfig 实际定义

**源码位置**: F2 从 `@antv/f-engine` 导出 coord 相关类型

```typescript
interface CoordConfig {
  type?: 'rect' | 'polar';  // 坐标系类型
  transposed?: boolean;      // 是否转置
  startAngle?: number;       // 起始弧度（仅极坐标）
  endAngle?: number;         // 结束弧度（仅极坐标）
  innerRadius?: number;      // 内半径（仅极坐标）
  radius?: number;           // 外半径（仅极坐标）
}
```

### 3.2 坐标系类型对照表

新增坐标系类型对照表：

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| `rect` | 直角坐标系，由 x、y 两个互相垂直的坐标轴构成 | 柱状图、折线图、散点图等 |
| `polar` | 极坐标系，由角度和半径两个维度构成 | 饼图、玫瑰图、雷达图等 |

### 3.3 CoordConfig 类型定义

新增完整的 TypeScript 类型定义：

```typescript
interface CoordConfig {
  type?: 'rect' | 'polar';
  transposed?: boolean;
  startAngle?: number;
  endAngle?: number;
  innerRadius?: number;
  radius?: number;
}
```
