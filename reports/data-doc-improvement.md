### 3.2 数据格式要求表

新增数据格式要求对照表：

| 要求 | 说明 |
|------|------|
| **数组格式** | 数据源必须是数组 |
| **对象元素** | 数组元素必须是对象 |
| **字段名** | 对象的键作为字段名，用于映射到图表属性 |
| **字段值** | 支持字符串、数字、数组、日期等类型 |

### 3.3 特殊图表数据格式

新增两种特殊图表数据格式：

**股票图（K线图）**:
```jsx
const data = [
  { date: '2023-01', open: 100, close: 110, high: 120, low: 95 },
];
```

**散点图（气泡图）**:
```jsx
const data = [
  { x: 10, y: 20, size: 5, category: 'A' },
];
```

### 3.4 数据处理方法

新增四种数据处理方法：

1. **数据过滤**: 使用 `filter()` 方法
2. **数据排序**: 使用 `sort()` 方法
3. **数据聚合**: 使用 `forEach()` + `Object.entries()`
4. **数据转换**: 使用 `map()` 方法

### 3.5 数据更新方法

新增动态数据更新方法：

```jsx
chart.changeData(newData); // 自动触发动画
```

### 3.6 常见问题处理

新增三种常见问题的处理方法：

| 问题 | 解决方案 |
|------|----------|
| 空数据 | 显示空状态组件 |
| 缺失值 | 过滤或填充默认值 |
| 大数据量 | 数据抽样、分页、服务端聚合 |

### 3.7 数据源类型

新增三种数据源类型说明：

| 类型 | 说明 | 示例 |
|------|------|------|
| 静态数据 | 直接定义在代码中 | `const data = [...]` |
| API 数据 | 从远程 API 获取 | `fetch('api/data')` |
| 用户输入 | 响应用户交互 | `chart.changeData(data)` |
