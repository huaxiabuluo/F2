## 二、类型定义分析

### 2.1 ScaleConfig 实际定义

**源码文件**: `packages/f2/src/scale/types.ts`

```typescript
export interface ScaleConfig {
  type?: 'linear' | 'cat' | 'time' | 'log' | 'pow';
  min?: number;
  max?: number;
  tickCount?: number;
  tickInterval?: number;
  mask?: string; // for time scale
}
```

### 2.2 CoordConfig 实际定义

**源码文件**: `packages/f2/src/coord/types.ts`

```typescript
export interface CoordConfig {
  type?: 'rect' | 'polar' | 'helix';
  innerRadius?: number; // for polar
  outerRadius?: number; // for polar
}
```

### 3.2 图形语法流程图

新增图形语法组成流程图：

```
数据 (Data)
  ↓
度量 (Scale)
  ↓
几何标记 (Geometry) + 图形属性 (Attribute)
  ↓
坐标系 (Coordinate)
  ↓
辅助元素 (Axis, Legend, Tooltip, Guide)
```

### 3.3 几何标记对照表

新增几何标记类型与图表类型的对应关系：

| 几何标记 | 组件 | 图表类型 |
|----------|------|----------|
| Interval | `<Interval />` | 柱状图、条形图、直方图 |
| Line | `<Line />` | 折线图、曲线图 |
| Point | `<Point />` | 散点图、点图、气泡图 |
| Area | `<Area />` | 面积图、区间图 |
| Candlestick | `<Candlestick />` | 蜡烛图（K线图） |

### 3.4 图形属性说明

新增图形属性详细说明：

| 属性 | 说明 | 示例 |
|------|------|------|
| `position` | 位置映射 | `x="genre", y="sold"` |
| `color` | 颜色映射 | `color="genre"` 或函数形式 |
| `size` | 大小控制 | `size={10}` 或函数形式 |
| `shape` | 形状控制 | `shape="circle"` |

### 3.5 度量类型

新增度量类型对照表：

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| `linear` | 线性度量 | 连续数值型数据 |
| `cat` | 分类度量 | 分类数据 |
| `time` | 时间度量 | 时间日期数据 |
| `log` | 对数度量 | 指数级增长数据 |
| `pow` | 指数度量 | 需要强调差异的数据 |

### 3.6 坐标系类型

新增坐标系类型对照表：

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| `rect` | 直角坐标系（默认） | 柱状图、折线图、散点图等 |
| `polar` | 极坐标系 | 饼图、玫瑰图、雷达图等 |
| `helix` | 螺旋坐标系 | 特殊可视化场景 |

### 3.7 辅助元素说明

新增辅助元素对照表：

| 组件 | 说明 |
|------|------|
| `Axis` | 坐标轴，显示数据刻度和标签 |
| `Legend` | 图例，标定不同数据类型 |
| `Tooltip` | 提示框，显示详细数据信息 |
| `Guide` | 辅助标记，添加辅助线、文本等 |

### 3.8 代码示例

新增 10 个实用代码示例：

1. **数据格式示例**: 展示数据源格式
2. **几何标记示例**: 柱状图、折线图、散点图、面积图
3. **图形属性示例**: 颜色映射、大小映射、形状映射
4. **度量配置示例**: 线性、分类、时间度量
5. **坐标系配置示例**: 直角坐标、极坐标、玫瑰图
6. **辅助元素示例**: 坐标轴、提示框、图例
7. **完整示例**: 可直接运行的完整代码
8. **图形语法映射表**: 说明各元素的作用
